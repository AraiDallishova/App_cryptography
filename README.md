# AES Encryption Project

## Overview
This project is a full Python implementation of AES encryption with ECB, CBC, CTR, and GCM modes.

## Features
- Encrypt and decrypt using AES
- Supports multiple modes (ECB, CBC, CTR, GCM)
- PKCS#7 padding
- AES-GCM with authentication tag
